#### 技术架构

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Node.js + Express
- **数据库**: JSON 文件存储 (轻量级)
- **API**: RESTful 风格

#### API 端点

```
用户相关:
  POST   /api/user/init       - 初始化用户
  GET    /api/user/:id        - 获取用户信息
  PUT    /api/user/:id        - 更新用户信息

健康日志:
  GET    /api/logs/today      - 获取今日日志
  PUT    /api/logs/:id        - 更新日志
  GET    /api/logs/history    - 获取历史日志

饮食记录:
  POST   /api/food            - 添加饮食记录
  GET    /api/food/today      - 获取今日饮食

AI 聊天:
  POST   /api/chat            - 发送消息

建议与报告:
  GET    /api/suggestions/today - 今日建议
  GET    /api/report/weekly   - 周报
  GET    /api/home            - 首页数据
```

#### 数据模型

```json
{
  "users": [],        // 用户基础信息
  "userGoals": [],    // 健康目标
  "dailyLogs": [],    // 每日健康日志
  "foodRecords": [],  // 饮食记录
  "chatSummaries": [],// 聊天记录
  "userTags": [],     // 用户画像标签
  "weeklyPlans": [],  // 周计划
  "aiSuggestions": [] // AI 建议历史
}
```
