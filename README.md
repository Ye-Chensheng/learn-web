# 🦞 vibe-healing

> 你的健康陪伴搭子 —— 不是帮你短期减肥，而是帮你在高压工作与有限时间里，维持一种可持续的健康生活方式。

## 📖 产品介绍

vibe-healing 是一个 **AI 驱动的健康陪伴系统**，专为年轻打工人设计。

### 核心理念

- **不是** 冷冰冰的打卡工具
- **不是** 制造焦虑的减肥应用
- **是** 理解你现实处境的健康搭子
- **是** 会根据你状态调整要求的智能助手

### 核心特性

| 特性 | 说明 |
|------|------|
| 🤖 AI 陪伴聊天 | 像朋友一样聊天，理解你的状态和情绪 |
| 📊 健康记录 | 轻量记录饮食、运动、睡眠、心情 |
| 🎯 动态建议 | 根据工作强度自动调整计划（保底/稳态/进阶） |
| 📈 周报反馈 | 每周总结，看到自己的进步 |

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

访问 `http://localhost:3000` 即可使用。

## 📊 数据模型

### 核心数据表

- `users` - 用户基础信息
- `user_goals` - 健康目标
- `daily_logs` - 每日健康日志
- `food_records` - 饮食记录
- `chat_summaries` - 聊天记录
- `user_tags` - 用户画像标签
- `weekly_plans` - 周计划
- `ai_suggestions` - AI 建议历史

## 🎨 设计理念

### 陪伴感

- 温和、不说教的语气
- 理解用户现实处境
- 允许低谷期存在
- 奖励稳定而非极端

### 低门槛

- 快捷记录，一键完成
- 聊天式交互
- 不强制每日打卡
- 支持"保底模式"
